Detailed Description

Singleton registry of USD capability declarations.

Capabilities are declared in plugInfo.json files under a "Capabilities" key in the "Info" section of a plugin:

{
"Plugins": [{
"Info": {
"Capabilities": {
"com.example.myCapability": {
"docstring": "Description of the capability",
"predecessors": ["usd"],
"isProfile": false
}
}
}
}]
}

Predecessors define the DAG edges: a capability implicitly satisfies all of its (transitive) predecessors. A capability with "isProfile": true is also accessible as a profile through UsdProfilesProfileRegistry.

The registry is populated lazily on first access and is read-only thereafter (thread-safe for concurrent reads).
