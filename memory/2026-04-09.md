### capabilities.json v2 — The Big Win
- Jens asked: can a browser app query profiles without OAV/enums?
- Answer: no — capabilities.json had DAG-only, metadata was in generated Python
- Solution: enrich capabilities.json with full requirement metadata
- This eliminates the need for generated Python enums entirely
- Mirrors Pixar's plugInfo.json → registry → validators pattern
